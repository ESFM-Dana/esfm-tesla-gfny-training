### How to add files
1. In GitHub, open **assets → files → <role>/** (e.g., `assets/files/frontline/`).
2. **Add file → Upload files** → drag in your PDF/DOCX/HTML.
3. **Commit changes**.
4. If filenames differ, update the link text above to match exactly.
